# Organología de la Pandereta Galega

La pandereta es un instrumento de percusión bimembranófono de marco. Su construcción y dimensiones son críticas para obtener el timbre característico.

## Especificaciones Técnicas

### Modelo Profesional

- **Diámetro**: 250mm
- **Marco (Aro)**: 68mm de altura, madera de haya o fresno.
- **Parche**: Piel de cabra de alta calidad, afinada para alta tensión.
- **Ferriños**: Sonajas de latón o acero dispuestas en pares.

### Modelo de Iniciación

- **Diámetro**: 220mm
- **Marco**: Madera ligera, apta para aprendizaje.
- **Parche**: Piel seleccionada para facilitar la emisión sonora.

## Mantenimiento

Es fundamental proteger el instrumento de la humedad extrema para preservar la tensión de la piel.
